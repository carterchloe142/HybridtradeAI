# 页面设计说明：账户侧栏/弹窗 + 自动更新洞察

## 全局设计（适用于本次改动涉及区域）
### Layout
- 桌面优先：整体采用 **CSS Grid + Flex 混合**。
- 顶部导航/头部区域：Flex 横向排列（左侧品牌/标题，中间可选搜索/状态，右侧用户资料入口）。
- 内容区：Grid 两列（主内容为洞察信息流；若存在右侧栏则固定宽度）；当账户侧栏打开时，主内容宽度自适应缩小或保持不变并覆盖（取决于你现有布局）。

### Meta Information
- Title: HYBRID TRADE AI - 洞察
- Description: 市场与新闻驱动的自动更新洞察与账户信息查看
- Open Graph: 与现有站点规则一致（沿用现状，不新增字段）

### Global Styles（Design Tokens）
- 背景色：#0B0F19（深色背景）或沿用现有主题
- 文本色：主文本 #E5E7EB；次级文本 #9CA3AF
- 强调色：#4F46E5（按钮/链接/高亮）
- 字体层级：
  - H1 24/32, H2 18/28, Body 14/20, Caption 12/16
- 按钮：
  - Primary：强调色底 + 白字；hover 提亮 6–10%
  - Ghost：透明底 + 边框/文本强调色；hover 加浅背景
- 链接：默认强调色；hover 下划线
- 动效：
  - 侧栏/弹窗打开：200ms ease-out（translate/opacity）
  - 列表更新：150ms 淡入（避免频繁闪烁）

## 页面：主界面（仪表盘/洞察）
### Page Structure
- 结构为“顶部导航 + 内容区（洞察信息流）”的堆叠布局。
- 洞察信息流使用卡片列表（单列）或两列卡片网格（取决于你现有信息密度；桌面优先建议单列更利于阅读）。

### Sections & Components
#### 1) 顶部导航（Top Nav）
- 左侧：产品名/Logo（沿用现状）。
- 右侧：**用户资料入口（可点击 div）**
  - 内容：头像（如有）+ 昵称/用户标识（如有）。
  - 交互：hover 提示可点击；点击打开“账户详情侧栏/弹窗”。
  - 可点击范围：整个 div（不是仅 icon/文字）。

#### 2) 洞察信息流（Auto-updating Insights Feed）
- 头部：
  - 标题“最新洞察”
  - 右侧小字显示：上次更新（例如“更新于 14:32”）
- 列表项（Insight Card）：
  - 标题（1 行，超出省略）
  - 摘要（2–4 行，超出省略）
  - 来源行：来源类型（市场/新闻）+ 来源名 + 发布时间
  - 可选：外链（sourceUrl 存在时显示“查看来源”）
- 状态：
  - 首次加载：骨架屏 3–5 条
  - 拉取失败：在列表上方显示轻量错误条（不遮挡整个页面）
  - 去重：相同 dedupeKey 的项不重复渲染；更新时仅增量插入/替换

#### 3) UI 清理（移除指定 span/svg）
- 目标：移除你已标记的 span/svg（例如装饰性分隔、重复 icon、冗余包裹节点）。
- 设计约束：
  - 移除后保持文本对齐与间距一致（使用统一的 gap/space token）。
  - 不改变交互命中区域（尤其是用户资料入口与洞察卡片的可点击区域）。

## 组件：账户详情侧栏/弹窗（Account Details Modal/Side Panel）
> 该组件由“用户资料入口点击”触发，覆盖在主界面之上。

### Layout
- 推荐桌面：**右侧侧栏（Side Panel）**
  - 宽度：360–420px
  - 位置：右侧贴边，垂直全高
  - 背景：与页面背景区分（例如 #111827）
- 备选：居中弹窗（Modal）
  - 宽度：480–640px
  - 垂直居中，带遮罩

### Structure
1. Header
   - 标题：账户详情
   - 关闭按钮：右上角“X”
2. Body（只读信息区）
   - 以定义良好的“键值行/信息块”展示现有账户字段：
     - 昵称/用户名（如有）
     - 邮箱/用户标识（如有）
     - 订阅/权限信息（如现有系统已提供）
   - 加载态：字段灰条骨架
   - 空态：提示“暂无可显示的账户信息”
3. Footer（可选）
   - 若你现有产品已有相关动作按钮则沿用；本次不新增动作。

### Interaction States
- 打开：点击顶部用户资料 div。
- 关闭：点击关闭按钮或遮罩；Esc 关闭（桌面友好）。
- 焦点管理：打开后焦点进入侧栏；关闭后返回到触发的用户资料入口。

### Responsive（桌面优先补充）
- >= 1024px：右侧侧栏。
- < 1024px：切换为全屏抽屉
